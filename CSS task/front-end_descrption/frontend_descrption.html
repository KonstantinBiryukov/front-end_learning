Содержание курса:

В курсе веб-разработки мы познакомим вас с клиенсткой веб-разработкой, которая включает в себя верстку - обучение HTML и CSS, и написание кода на JavaScript. На курсе JavaScript будут рассмотрены синтаксис, типы данных, работа с DOM и событиями, стандартная библиотека языка.

В курсе JavaScript также рассматривается большое количество библиотек и средств разработки - jQuery, jQuery UI, Underscore, Vue.js, npm, Bootstrap, Webpack, LESS/SASS, Babel. Также обзорно рассматриваютя Grunt, RequireJS, TypeScript. В качестве серверной части используем Express.js. Обучение веб-разработке закрепляется на практике.

Вы научитесь создавать клиентскую часть сайта и получите широкий кругозор по клиентским технологиям.
